Getting Started

Welcome to SOLUS

SOLUS is a powerful order analytics application designed for tracking retail orders across multiple stores. It supports two distinct data import methods — each optimized for different workflows.

📧 IMAP Mode

Connect email accounts to sync order confirmations, shipping updates, and delivery notifications directly from your inbox.

💬 Discord Mode

Link with the SOLUS Discord bot to import orders from ACO and similar automation tools that post to Discord channels.

Supported Retailers

Walmart Target Pokemon Center Best Buy Costco Sam's Club

First Launch

1

Activate Your License

Enter your license key (format: SOLUS-XXXX-XXXX-XXXX) and click Activate.

2

Choose Your Data Mode

Select 📧 IMAP or 💬 Discord based on how you want to import orders.

3

Connect Your Data Source

Add email accounts (IMAP) or link your Discord account (Discord mode).

Data Modes Explained

SOLUS operates in one of two exclusive modes. The entire interface adapts based on your selected mode, showing only relevant features and hiding those that don't apply.

📧 IMAP Mode

Best for: Direct email account access, full order lifecycle tracking

  • Full tracking — Confirmed → Shipped → Delivered
  • Carrier info — FedEx, UPS, USPS, DHL tracking
  • Delivery addresses — Full address extraction
  • Prices — Order totals from confirmation emails
  • Deliveries page — Track packages by address
  • Account Stats — Group by email address

💬 Discord Mode

Best for: ACO bot users, profile-based tracking

  • Confirmations & cancels — From bot posts
  • Profile tracking — Group by bot profile name
  • Bot type badges — See which bot placed each order
  • Real-time sync — Auto-sync from watched channels
  • Profile Stats — Performance by profile
  • Streamlined UI — No shipping/delivery clutter
💡

Which Mode Should I Use?

Use IMAP if you want complete order tracking including shipping and delivery. Use Discord if you primarily use ACO or similar bots and want to track performance by profile name.

Switching Between Modes

1

Go to Settings

Click the gear icon in the sidebar navigation.

2

Find Data Mode

Under the General tab, locate the "Data Mode" option.

3

Select Your Mode

Choose Email (IMAP) or Discord. The UI instantly updates.

ℹ️

Your Data is Safe

Switching modes doesn't delete anything. Orders from both sources are stored separately and will reappear when you switch back.

Discord Bot Setup

💬

Discord Mode Feature

This section applies when using Discord mode. The Discord page appears in the sidebar only when Discord mode is active.

The SOLUS Discord integration allows you to import orders from automation bots (like ACO) that post order confirmations to Discord channels. The bot watches your specified channels and relays order data to SOLUS.

How It Works

Architecture Overview
  1. You invite the SOLUS bot to your Discord server
  2. You link your SOLUS license to your Discord account using a one-time token
  3. You tell the bot which channels to watch (where ACO posts orders)
  4. The bot captures order posts and stores them in the cloud
  5. SOLUS syncs orders from the cloud to your local app

Step 1: Invite the Bot

1

Go to Discord Page

In SOLUS, click Discord in the sidebar (requires Discord mode).

2

Click "Invite Bot to Server"

Opens Discord's authorization page in your browser.

3

Select Your Server

Choose the server where your order bot (ACO) posts. You need "Manage Server" permission.

4

Authorize Permissions

The bot needs permission to read messages in channels you specify.

Step 2: Link Your Account

1

Generate Link Token

In SOLUS Discord page, click "Generate Token". A 6-character code appears.

2

Use /link Command in Discord

In any channel where the bot is present, type: /link [TOKEN]

3

Confirm Link

The bot confirms the link. Your Discord username now appears in SOLUS.

⏱️

Token Expires in 5 Minutes

Link tokens are single-use and expire quickly for security. Generate a new one if it expires.

Step 3: Watch Channels

Tell the bot which channels contain order posts from your automation tools.

Discord Commands
CommandDescription
/link [token]Link your Discord to SOLUS license
/unlinkDisconnect your Discord from SOLUS
/watchStart monitoring the current channel for orders
/unwatchStop monitoring the current channel
/scan [hours]Scan channel history for past orders (default: 24 hours)
/statusShow link status and watched channels
/theme [color]Change embed color theme (purple, blue, green, red, orange, pink)
1

Go to Your Order Channel

Navigate to the Discord channel where ACO (or your bot) posts order confirmations.

2

Type /watch

The bot confirms it's now watching this channel.

3

Repeat for Other Channels

Watch multiple channels if you have orders posting to different places.

💡

Import Historical Orders

Use /scan 48 to scan the last 48 hours of channel history for orders you might have missed. The bot will parse and import any order posts it finds.

Step 4: Sync Orders

Once channels are watched, the bot captures new orders automatically. Sync them to SOLUS:

SettingDescription
Auto-SyncToggle automatic syncing on/off
Sync IntervalHow often to check (5, 15, 30, or 60 minutes)
Sync NowImmediately sync new orders
Refresh ImagesRe-match products to local SKU images
Clear Discord OrdersRemove all Discord orders (keeps email orders)

ACO Bot Support

The SOLUS bot is designed to parse order posts from ACO (Automated Checkout) and similar bots. It extracts:

  • Profile Name — Which bot profile placed the order
  • Bot Type — The automation tool used (shown as a badge)
  • Retailer — Walmart, Target, etc.
  • Order ID — The retailer's order number
  • Item Name — Product ordered
  • Status — Confirmed or Cancelled
💡

Profile Tracking

Discord mode excels at tracking performance by profile. See which profiles have the best stick rates in Profile Stats.

Troubleshooting

IMAP Issues

"Authentication failed"
  • Use App Password, not regular password
  • Enable 2FA for Gmail
  • Enable IMAP in email settings

Discord Issues

"Link token expired"
  • Tokens expire in 5 minutes
  • Generate a new token and use immediately
No orders syncing
  • Verify you've watched the correct channels with /watch
  • Check bot has permission to read messages
  • Enable auto-sync or click "Sync Now"
  • Make sure you're in Discord mode (not IMAP mode)
Bot not responding to commands
  • Ensure bot was properly invited to server
  • Check bot has permission to use slash commands
  • Try in a different channel

Missing Images

Orders showing placeholder images
  • Use the SKU Mapper tool to map product names to SKUs
  • Click "Refresh Images" in IMAP or Discord page
  • Ensure matching images exist in product-images/ folder
